KAPLAY es una librería de JavaScript que te ayuda a hacer juegos de forma rápida y divertida!

Empieza a probarlo en el [KAPLAYGROUND](https://play.kaplayjs.com)

## Características

- **Fácil y Divertido**: La API de KAPLAY es fácil y divertida de usar.
- **Diseñado para Juegos**: KAPLAY está diseñado desde cero para hacer juegos, no solo renderizar gráficos.
- **Librería**: KAPLAY es una librería, no un framework. Puedes usar tanto o tan poco como quieras.
