## Instalación Recomendada 

Una vez que tengas [Node.js](https://nodejs.org/en/download) instalado, la forma más fácil de comenzar con **KAPLAY** es usar
[`nuestro paquete de instalación`](https://www.npmjs.com/package/create-kaplay), que
generará para ti utilizando la estructura **recomendada**.

<JustForVersion version="3001">
```sh
npx create-kaplay myGame 
```
</JustForVersion>


<JustForVersion version="4000">
```sh
npx create-kaplay myGame --version next
```
</JustForVersion>

Luego, utilizando `npm run dev` puedes iniciar el **servidor de desarrollo**, donde
puedes probar tu juego localmente en [`https://localhost:5173`](https://localhost:5173).

```sh
cd myGame && npm run dev
```

Eso es todo! Estás listo para **empezar a hacer juegos**. Si quieres conocer todas las opciones
disponibles en el paquete `create-kaplay`, ejecuta `npx create-kaplay --help`.

## Instalación Sin Empaquetadores 

Si prefieres usar **KAPLAY** **sin ningún empaquetador**, ¡puedes hacerlo! Comienza con
un archivo `index.html`:

```html
<html>
  <head>
    <title>My Game</title>
  </head>
  <body></body>
</html>
```

<Info crew="kat" title="Serve your statics!">
Para usar correctamente el CDN y HTML, no puedes simplemente abrir el `index.html`, necesitas
servir tus archivos. Ejecuta `http-server`:

```sh
npm install --global http-server
```

```sh
http-server # Sirve tus archivos 
```

</Info>

## Instalación custom con Node.js 

Así que, ¿Eres un profesional? Vamos a crear nuestra propia plantilla usando Node.js. Primero,
necesitamos instalar la biblioteca KAPLAY en una carpeta nueva:

```sh
npm install kaplay
```

Tendrás que usar un empaquetador para usar KAPLAY con NPM. Hay muchas opciones
como `esbuild`, `webpack`, `parcel`, `vite`, etc. Este es un breve ejemplo usando [`esbuild`](https://esbuild.github.io/).

```sh
npm install esbuild
```

Una vez que tengas `esbuild` instalado, y tengas esto en un archivo `.js` o `.ts`:

```js
import kaplay from "kaplay";

kaplay();
```

solo ejecuta

```sh
esbuild game.js --bundle > build.js
```

y encontrará el paquete KAPLAY e lo incluirá en el `build.js` compilado,
incluye `build.js` en tu HTML y estarás listo para comenzar. Siéntete
libre de automatizar este proceso.
